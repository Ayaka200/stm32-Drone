# stm32-Drone
基于stm32Hal库和FreeRTOS操作系统的四轴无人机程序

项目简介

本项目为独立设计开发的四轴无人机飞控系统，完成了飞控板硬件设计、PCB制作、硬件焊接及软件开发全流程。系统采用 STM32F103 作为主控芯片，FreeRTOS 作为实时操作系统，结合 MPU6050 实现姿态解算与飞行控制，并通过 NRF24L01 实现无线遥控通信。

项目实现了姿态稳定、遥控控制、电压监测、失联保护等功能，可完成四轴无人机的基本飞行控制任务。

主要功能

- MPU6050 六轴传感器数据采集
- 卡尔曼滤波姿态解算
- PID姿态控制
- FreeRTOS多任务调度
- NRF24L01无线遥控通信
- OLED实时数据显示
- 电池电压监测与回传
- 遥控器OLED状态显示
- 失联保护自动缓降
- PWM电机驱动控制

硬件平台

飞控端

- STM32F103C8T6
- MPU6050
- NRF24L01
- OLED显示模块
- 锂电池供电系统

遥控端

- STM32F103C8T6
- NRF24L01
- OLED显示模块
- 双摇杆输入模块

软件架构

系统基于 FreeRTOS 实现任务管理：

- IMU数据采集任务
- 姿态解算任务
- PID控制任务
- NRF通信任务
- OLED显示任务
- 按键输入任务

项目亮点

1. 完整独立开发

独立完成：

- 原理图设计
- PCB设计
- 硬件焊接与调试
- STM32底层驱动开发
- FreeRTOS移植与应用
- 飞控算法调试

2. 实时控制系统

采用 FreeRTOS 实现多任务管理，通过任务优先级设计保证姿态控制实时性。

3. 安全保护机制

实现失联检测功能，当遥控器通信中断时，无人机自动降低油门并缓慢下降，提高飞行安全性。

开发环境

- STM32CubeMX
- CLion
- Keil MDK
- FreeRTOS
- STM32 HAL Library
